Gerald de Jong (Delving B.V.)

Tweet

Gerald has a mathematics background and has been writing code since he was a teenager. After some years of building software in C and C++ he switched to Java when it appeared in 1996, and became a trainer for Sun Microsystems. Since then, his one-man company Beautiful Code BV has been helping clients with design, training, and coding. In the last few years his focus has been in the domain of Cultural Heritage (Museums, Archives, Libraries) with a collaboration called Delving BV, in Java, Scala, and now also Javascript/AngularJS. He loves the richness, diversity, and potential of cultural heritage data.

A Sophisticated AngularJS Cultural Heritage Application

Room: lightroom Time: 16:10 - 16:50
Language: English Level: Intermediate

For the provinces of Brabant and Friesland my company (Delving) has deployed our open source cultural heritage software infrastructure, and we recently built an application for gathering data from online contributors. The focus is on Historical Societies (Heemkundekringen) which made it an excellent usability challenge (these are older people). AngularJS revolutionized our development, because it's the first Javascript framework which made great sense to me, a dyed-in-the-wool Java coder from way back. I would like to give demonstrations and code walk-throughs and give feedback on how it was done.
